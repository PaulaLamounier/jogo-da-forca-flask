# 🎯 Jogo da Forca com Flask

## 📌 Sobre o projeto

Aplicação web desenvolvida em Python utilizando Flask, com o objetivo de praticar lógica de programação e conceitos básicos de desenvolvimento web.

---

## 🎮 Funcionalidades

* Geração de palavra aleatória
* Controle de tentativas
* Exibição do estado do jogo
* Interação com o usuário via interface web

---

## 🛠️ Tecnologias utilizadas

* Python
* Flask
* HTML
* CSS

---

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

2. Instale as dependências:

```bash
pip install flask
```

3. Execute o projeto:

```bash
python app.py
```

4. Acesse no navegador:

```bash
http://localhost:5000
```

---

## 📁 Estrutura

* `app.py` → lógica da aplicação
* `templates/` → páginas HTML
* `static/` → arquivos estáticos (CSS, imagens)
